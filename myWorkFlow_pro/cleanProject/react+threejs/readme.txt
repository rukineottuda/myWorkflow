Это стартовая болванка для запуска проекта на React.js
+подключен  фреймворк Three.js, для  взаимодействия с WebGL
+установлен bootstrap

1)npm install
2)npm start
запустится сервер, с react и уже подключенным модулем three.js(<_ThreeScene />)
Если Threejs не требуется можно отключить.И будет чистый React.
