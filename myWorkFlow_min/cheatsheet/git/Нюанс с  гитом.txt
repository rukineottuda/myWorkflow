Если у кого-то не ставиться плагин superfish, то значит у Вас не установлен Git
Чтобы все устранить, нужно
1) Скачать Git для Windows в гугле
2) Установить, там будет много окон, нажимаем далее, пока не дойдем до этапа установки
3) После того, как установили, на диске C в папке где вы установили, скорей всего Program Files будет папка Git
4) Открываем папку Git
5) там будут две папки bin и cmd, они нам нужны будут
6) На рабочем столе на мой компьютер нажимаем правой клавишой мышки, выбираем свойства, потом Расширенные свойства системы, дальше появляется окно, внизу окна должна быть кнопка "Переменные среды", нажимаем ее
7) будет переменная с именем PATH, нажимаем на нее, нажимаем кнопку редактировать, в значение переменной дописываем значение ";C:\Program Files\Git\bin;C:\Program Files\Git\cmd" , у меня было установлено в папку Program Files, у вас может быть в другую, точка с запятой отделяет пути, первую нужно писать перед первым значением
8) Открывем строку, пишем Git, если появилось окно, то значит установлен
9) Теперь как в видео, в нужной папке с проектом, нажимаем правой клавишой по папке запуск командной строки сдесь, пишем "bower i superfish" и он должен установить плагин в папку app/libs
10) Все можно изучать урок дальше
